Specifications

1. The application must support playback of ‘.mkv’ and ‘.mp4’ video files.
2. The application must support audio playback from ‘.mkv’ and ‘.mp4’ files.
3. Both the videos and the application must function seamlessly across all devices and be responsive (TV, PC, Android).
4. The application must support .vtt and .srt subtitles for videos. (Please note that it is mandatory to use the same name for movie and for subtitles in order to be able to load them in the application)
4. The application must be easy to install and user-friendly for all users. *

Note:
* This feature is still under development.
